# LibraryPro - Book Management Dashboard 📚✨

A modern, premium Web Engineering project built to manage book collections effortlessly. This platform features robust PHP/MySQL backend logic combined with a visually stunning, glassmorphism-inspired UI and an interactive 3D background.

## 🌟 Key Features

- **Full CRUD Operations:** Seamlessly add, view, update, and delete book records.
- **Premium UI/UX:** Designed with a stunning "Glassmorphism" aesthetic using custom CSS properties, smooth hover effects, and sleek translucent cards.
- **Interactive 3D Background:** Integrated Vanta.js and Three.js to create a dynamic, animated background that responds to user interaction.
- **Real-Time Dashboard:** Dynamic statistics calculating total books and overall collection value directly from the database.
- **Fully Responsive:** Styled with Bootstrap 5.3 to ensure it looks flawless on any device.

## 🛠️ Tech Stack

- **Backend:** PHP, MySQL
- **Frontend:** HTML5, Vanilla CSS3 (Glassmorphism), Bootstrap 5.3
- **Libraries:** Vanta.js, Three.js, FontAwesome 6.4

## 🚀 Setup Instructions

1. Clone the repository.
2. Import the `database.sql` into your MySQL database.
3. Update `db_connect.php` with your local database credentials if necessary.
4. Launch the project via a local server (e.g., XAMPP, WAMP, or Docker).
